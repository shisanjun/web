##本节作业


###作业：后台管理平台

####要求：
	后台管理平台 ，编辑表格：
	1. 非编辑模式：
	可对每行进行选择； 反选； 取消选择
	2. 编辑模式:
	进入编辑模式时如果行被选中，则被选中的行万变为可编辑状态，未选中的不改变
	退出编辑模式时，所有的行进入非编辑状态
	处于编辑模式时，行被选中则进入编辑状态，取消选择则进入非编辑状态
