###1. 程序说明
	电子商务网站页面（动态）（仿京东）
	主页实现动态效果有
	1）顶部广告位关闭
	2）头部二维码显示大二维码效果
	3）左边菜单导航，展示更多菜单效果
	4）中部大图广告位，移动下边选择按钮更换图片
	5）促销切换TAB页
	6）秒杀时间倒计时
	7）返回顶部
